# Budapest Időjárás Asszisztens

Egyszerű, böngészőből használható időjárás asszisztens Budapest kerületeire.

## Mit tud?
- 3 napos előrejelzés
- Budapest összes kerülete külön választható
- Időjárási riasztások (hőség, fagy, szél, csapadék)
- Öltözködési és kertészeti tippek
- Automatikus adatfrissítés
- API-kulcs és regisztráció nélkül működik

## Használat
1. Nyisd meg ezt a linket:
   https://milan4o4.github.io/idojaras-asszisztens/

3. Kész
